# ZigNav

Navigate and click UI elements without touching your mouse.

## Features

- **Global Hotkey Activation**: Press `Cmd+Shift+Space` to scan the frontmost app
- **System UI Mode**: Press `Cmd+Shift+Enter` to scan menu bar and Dock
- **Letter Labels**: Type letter labels (A-Z, AA-ZZ) to select elements
- **Keyboard Actions**:
  - `Return` - Click the selected element
  - `Shift+Return` - Right-click (show context menu)
  - `Escape` - Dismiss overlay
  - `Delete/Backspace` - Clear typed input
- **Menu Navigation**: Automatically continues through submenus

## Requirements

- macOS 12.3 (Monterey) or later
- Accessibility permission
- Input Monitoring permission

## Installation

### Option 1: Download Release

1. Download `ZigNav.app.zip` from the [Releases](https://github.com/your-repo/zignav/releases) page
2. Unzip and move `ZigNav.app` to `/Applications`
3. Right-click and select "Open" (required for unsigned apps)
4. Grant permissions when prompted
